# MeterMate Admin Favicon Files

This folder contains the favicon files for the MeterMate Admin panel. Please replace the placeholder files with your actual images:

## Required Files:

### favicon.ico
- **Size**: 16x16, 32x32, or 48x48 pixels
- **Format**: ICO
- **Purpose**: Traditional favicon for browsers

### favicon-16x16.png
- **Size**: 16x16 pixels
- **Format**: PNG
- **Purpose**: Small favicon for modern browsers

### favicon-32x32.png
- **Size**: 32x32 pixels
- **Format**: PNG
- **Purpose**: Standard favicon for modern browsers

### apple-touch-icon.png
- **Size**: 180x180 pixels
- **Format**: PNG
- **Purpose**: Icon for Apple devices when adding to home screen

### android-chrome-192x192.png
- **Size**: 192x192 pixels
- **Format**: PNG
- **Purpose**: Icon for Android devices (small)

### android-chrome-512x512.png
- **Size**: 512x512 pixels
- **Format**: PNG
- **Purpose**: Icon for Android devices (large)

## Design Guidelines:

- Use the MeterMate branding colors (blue theme: #2563eb)
- Ensure the icon is recognizable at small sizes
- Keep the design simple and clear
- Consider using a meter or house icon to represent the application
- Make sure the icon works well on both light and dark backgrounds

## How to Replace:

1. Delete the placeholder text files
2. Add your actual image files with the exact same names
3. Ensure the file formats and sizes match the requirements above
4. The favicon will automatically appear in browser tabs and bookmarks

## Testing:

After adding your images, you can test the favicon by:
1. Starting the development server: `npm run dev`
2. Opening the admin panel in your browser
3. Checking the browser tab for the favicon
4. Testing on different devices and browsers
